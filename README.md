# downloadByGSpread
Сборка отчётов и базы данных по отчётам администраторов:

Кое-где в гугл-диске хранятся отчёты сотрудников компании.
Данная программа умеет следующие вещи:
 - Сделать выгрузку отчётов за указанный интервал
 - Выгрузку базы контактов из отчётов сотруднков для её дельнейшего хранения
 - Создать сводку про заполенние отчётности администраторов
 - Получить список всех внесённых в систему администраторов
 
Каждая выгрузка отчёта создаёт папку с форматом: report{дата, когда получен запрос}
Основной отчёт - {[дата1][дата2]}dailyReport. Он, даже, цветной!
